1. HTML(HYPERTEXT MARKUP LANGUAGE) - 링크걸어주는 텍스트
- HYPERTEXT: HTML 핵심 기능 중 하나로 웹 페이지간의 링크(하이퍼링크)를
통해 다른 문서나 리소스에 연결할 수 있도록 해주는 텍스트를 말한다. 
- MARKUP: 웹 페이지의 구조를 정의하기 위한 특수한 태그나 속성을 말한다.사전적으로는 '원가에 붙여지는 마진' 정도로 해석할 수 있다.
- HTML은 웹 브라우저를 통해 표시되는 웹 페이지의 구조와 컨텐츠를 정의하기 위해 사용되는 언어를 말한다. 
개발자가 HTML코드로 페이지에 표시될 내용을 정의하는 문서를 만든다. 
- 요약하면 하이퍼텍스트와 컨텐츠를 표시해주는 언어라고 말할 수 있다.

2. 렌더링(RENDERING) - 디자인한데로 웹브라우저가 보여주는 것
- 렌더링 : 컴퓨터 그래픽을 통하여 2,3차원 이미지를 그리는 것을 렌더링이라고 한다. 
- 개발자가 HTML 문서에 태그 또는 속성을 이용해서 페이지가 어떻게 보여질지에 대한 정의를 하면 웹 브라우저는 그
HTML문서를 읽어들여서(로딩해서) 해석하고 개발자가 구성한 대로 표현해준다. 이것을 "렌더링"이라고 한다ㅏ. 
쉽게 말하면 웹브라우저 화면에 무엇인가를 그린다고 생각하면 된다.

3. <!DOCTYPE HTML>구문 - 옛날 문서 좆까, 나 최신 HMTL5규약으로 문서 실행할꺼야
- Html5 문서임을 명시하는 선언이다. 
웹 브라우저에 표준 모드로 문서를 렌더링하도록 지시한다.
만약에 이 구문이 없을 경우 쿼크 모드로 실행된다. 쿼크 모드란 이전의 표준 규약으로 만들어진 문서와의
호환성을 유지할 수 있도록 해주는 모드이다.
그런데 이런 과거 표준들은 여러가지 문제를 일으킬 소지가 있다. 예를 들면 CSS해석이나 자바스크립트 실행시
문제를 야기시킬 수 있다.

4. HTML5 웹 표준이란?
-웹 페이지를 만드는 규약은 시간에 따라서 진화해왔다.
HTML5은 HTML의 최신 버전으로, 웹페이지 및 애플리케이션을 작성하는 데 사용되는 표준 마크업 언어를 정의하는
규약을 말한다. HTML5은 이전 버전인 HTML 4.01과 XHTML 1.0에 비해 여러가지 새로운 기능과 개선 사항을 도입하여
웹페이지를 더욱 강력하고 유연하게 만들 수 있게 지원한다.

5. <HTML LANG="KO"> 구문
- <HTML> 태그는 전체 HTML 문서를 감싸는 루트 요소(ELEMENT)이다.
- LANG : HTML 태그의 속성으로 문서의 주요 언어를 정의한다.
웹페이지의 주요 언어가 한국어(KO)임을 웹브라우저에게 알리는 역할을 한다.
이렇게 하면 웹 접근성과 SEO(검색엔진 최적화)에 중요한 영향을 미친다.

6. <HEAD> 섹션
- HTML문선의 메타 데이터를 정의하는 영역이다. 메타 데이터는 HTML 문서에 대한 다양한 정보로 
웹브라우저에게 직접 노출되지는 않지만 META, TITLE, LINK STYLE, SCRIPT 등의 태그를 통해서 
HTML 문서 표현에 대한 다양한 정보를 제공한다.
- <TITLE> 문서의 제목을 나타내며, 브라우저의 탭에 표시된다.
- <META CHARSET="UTF-8"> HTML 문서의 문자 인코딩 방식을 지정한다.
- <META NAME="VIEWPORT"> 반응형 웹페이지를 표시하기 위한 모바일 친화적인 설정을 제공한다.
예를 들면, 페이지를 스마트폰에서 열때와 PC에서 열때는 화면의 크기가 달라진다. 
디바이스의 크기에 맞춰서 화면이 자동으로 사이즈가 조절되도록 해주는 기능을 한다.

7. <BODY> 태그
- 문서의 본문(컨텐츠)를 감싸는 요소로 브라우저에 실제로 표시되는(보여지는) 모든 컨텐츠가 이 요소 안에 
들어간다. <BODY> 태그 안에다 정의한 내용은 모두 웹 브라우저에 보여진다.

8. 주석(COMMENT) //, --, /**/
- 브라우저에 의해서 해석되지 않는다. 즉 페이지에 보여지지 않는다. 
대신에 다른 개발자에게 이 HTML 코드를 분선하는 데 도움을 준다. 
- <!-- 주석 내용을 적고 --> : 주석의 문법

9. DTD(DOCUMENT TYPE DEFINITION)
- <!DOCTYPE HTML>과  같이 HTML문서의 유형을 정의하는 코드로 문서의 첫문장에 사용되어 브라우저에게 
문서의 유형을 알려주어 문서가 올바르게 웹페이지를 표시할 수 있게 해준다.

10. 태그(TAG)
- HTML문서를 구성하는 핵심요소이다. 하이퍼텍스트와 이미지 등이 웹브라우저에 표시될 수 있도록, 
정희하는 역할을 한다.

- <여는 태그 속성명 = "속성값"> 컨텐츠(화면에 표시될 내용)</닫는 태크>
